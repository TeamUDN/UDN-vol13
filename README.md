# Vue-Flask-practice

Vue CLI + Flask の練習

## Vue CLI の build
```
$ npm run build
```

## フロントエンド単体での開発用サーバ起動
```
$ cd frontend
$ npm run serve
```

## FlaskのWebサーバ起動
```
$ cd src
$ pipenv run start
```

## Docker起動
```
$ docker compose up
```
